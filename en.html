Scanlab is a deep web search engine that allows users to find a lot of different network devices in the internet. Scanlab was developed as an alternative to shodanhq, but became free and opensource software. You can install scanlab to your own server and do researches in the dark internet. You'll just need some linux skills and nmap.

Why Scanlab?

Scanlab will be useful for security specialists and freelance darknet researchers. All the port scanning magic we leave to nmap.

Scanlab builds a search engine from nmap XML reports. It is pretty useful when you have a lot of reports and need to search through them.

What is deep web?

Deep web is World Wide Web content that is not part of the Surface Web, which is indexed by standard search engines.

Everyday people buy new network devices and connect them to the internet. Those devices can be discovered and exploited from the global network. People are lazy, they always forget to change their default passwords. You can gain access to thousands devices like printers, routers, phones, cameras in the deep web.

Features

Usage of Scanlab is really easy: just log in and upload XML reports to a web form. Scanlab saves them to the database and builds the search engine.

Scanlab has unique features: device tags, rating, console client.

Device tags are device tags. Scanlab gives tags to reports, so you can find cameras, printers and other devices easily.

Rating and favorites are for sorting and saving interesting reports. Adding the report to favorites increases its rating.

Console client is useful for automation and uploading large reports.

Scanlab is free and opensource software, opposed to shodanhq.

Debian Wheezy, Kali Linux, Ubuntu:

Use the automatic installer. Go to your installation folder:

mkdir scanlab && cd scanlab

Run this as root:

curl -s http://scanlab.info.tm/installer | bash -

Complete the installation at http://localhost/install.php
